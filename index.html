This Very Site!

After looking at examples of other web developers' portfolios I decided to attempt to design my own custom front end portfolio site instead of using a template. I was eager to learn some additional CSS skills. Check out the repo on github!

FEEDbucket

Our 3rd project was a team-based assignment. We were assigned to both a team and a prompt and were told to use certain technologies for the backend - MongoDB, Mongoose, and Express. Check out the front end repo and the backend repo on github!

Gifter

For our 2nd project at GA we were tasked to create any full stack site. I created this site as a place for a user to save gift ideas for different people. Take a look at both the front end and back end repos.

Tic-Tac-Toe

Our very first project at GA was to create a front end site for a seemingly simple game of Tic-Tac-Toe. I was able to add in the extra challenge of allowing a user to view or even resume their prior games. You can take a peek at the github here!
